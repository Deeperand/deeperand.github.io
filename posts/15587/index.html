[TRICK] Common Used Shortcuts of Logic Pro

最后更新：2021.11.09, 10:22

This list just record the shortcuts I’ve used. The more detailed list can see [1]. Note

  • <M-> $\equiv$ <option->
  • <C-> $\equiv$ <control->
  • <S-> $\equiv$ <shift->

And the notation in Logic Pro shortcuts means

  • ⌤: Projective, <fn-enter> [2]

General Operation

Edit (General)

  • <cmd-c>: copy
  • <cmd-x>: cut
  • <cmd-v>: paste
  • <cmd-S-v>: “取代貼上”
  • <cmd-a>: select all
  • <cmd-z>/<cmd-S-z>: undo/redo
  • <cmd-M-z>: “檢視操作歷史紀錄”
  • T: “更換滑鼠工具”

Move/Play (General)

  • <space>: play/pause
  • <enter>/<return>: jump to start of the project
  • <.>:
  • <>
    快轉（以 1小節 為單位移動 Play Head）：。 (大於/句號)
    倒轉（以 1小節 為單位移動 Play Head）：， (小於/逗號)
    快轉（以 1小節 為單位移動 Play Head）：Shift + 。 (大於/句號)
    倒轉（以 1小節 為單位移動 Play Head）：Shift + ， (小於/逗號)
    啟動/關閉 圈選範圍 （Cycle）：C
    圈選目前選取的 Region 或是 MIDI Note 範圍 （Cycle）：Ｕ
    移動 Play Head 到指定位置： / （問號 / ㄥ）

Marker

  • <M-k>: show/hide markder track
  • <M-'>: Create Marker
  • <M-C-'>: Create Marker without rounding
  • <M-S-'>: Create Marker for Selected Regions
  • none: Create Marker by Cycle Area •
  • <M-delete>: Delete Marker
  • none: Open Marker List
  • none: Open Marker Text
  • <M-S-C>: Set Locators by Marker and Enable Cycle
  • <M-S-,>Set Locators by Previous Marker and Enable Cycle
  • <M-S-.>: Set Locators by Next Märker and Enable Cycle
  • none: Go to Previous Marker
  • none: Go to Next Marker
  • <M-,>Go to Previous Marker And Set Locators
  • <M-.>: Go to Next Marker And Set Locators
  • <M-/>: Go to Marker Number…
  • <S-'>Rename Marker

Window Control

Display/Close Main Window

  • Y: library
  • I: inspector
  • <cmd-M-T>: tool bar
  • B: smart control
  • X: mixer
  • E: audio track edit / piano roll / drummer
  • P: piano roll
  • G: global track
  • D: event list edit
  • O: apple loops
  • <M-C-P>: note pads
  • F: (media) browser
  • <M-k>: key command (shortcuts setting)

Open Single Window

  • <cmd-1>: open main window
  • <cmd-2>: open mixer
  • <cmd-3>: smart control
  • <cmd-4>: piano roll
  • <cmd-5>: score
  • <cmd-6>: audio file edit
  • <cmd-7>: event list
  • <cmd-8>: project audio browser
  • <cmd-9>: MIDI transformation
  • <M-E>: “event float”
  • <M-R>: region inspector
  • <cmd-K>: musical typing
  • <cmd=,>: preference
  • <cmd-P>: project setting

Piano Roll

(main ref[4])

  • Q: quantication
  • <S-click>: multi-select
  • <cmd-R>: repeat
  • ;: shift to playhead location
  • <S-F>: select following
  • <S-I>: inverse selection
  • <M-S-up>/<M-S-down>: move octave
  • <M-up>/<M-down>: move octave
  • <C-M>: mute
  • <S-\>: force legato
  • <\>: conceal overleap
  • <M-S->: assign operation for all the selected notes
    • <M-S-velocity tool>: assign the same velocity for all the selected notes [3]
    • <M-S-prolog notes>: prolong all the selected notes to the same length
