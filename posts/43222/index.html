General Knowledge about Font File

OTF and TTF

TTF is abbreviation of “True Type Fonts”, and OTF is abbreviation of “Open Type Fonts”. OTF is newer than TTF, and it is normally more powerful than TTF [3] [4] , so it should be considered at first.

Download Free Chinese Font

Source Han Sans (思源黑体) [1] is a nice open source sans-serif font, the meaning of its different version can be summarized by [2]

  • HW: half angle
  • VF: Variable Fonts (可变自重字体)
  • SC: Simplify Chinese
  • CN: Chinese
  • J: Japanese
  • K: Korea
  • HC: Hong Kong Chinese
  • TC: Taiwan Chinese

There is another font Noto Sans Simplified Chinese [5] public by Google, it is almost identical with Source Han Sans. I don’t know what difference between, and probably the only difference is the Noto one is a font brand of Google [7] and the Source one didn’t have that brand and can be found on GitHub.

Note the version cannot be Variable OTCs ! This version miss weights of the font (probably since this is a “variable” version so it no need to have all the weights?), and this will lead the assumption of using option [BoldFont=*-Medium] to adjust the weight of font failed.
